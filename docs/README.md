# 2023 12월 우테코 식당 프로모션 플래너
   
- **12월** 우테코 식당의 **🎄이벤트 프로모션 플래너** 입니다.    
- 식당에 방문 예정인 고객들은 이 플래너를 통해 본인에게 적용되는 **다양한 혜택 정보**를 확인해볼 수 있습니다.
- 플래너를 통해 많은 고객들이 우테코 식당에 방문하기를 기대합니다 :)  
     
<br>     

## 🎯 플래너 목표
- 고객들이 **혜택을 많이 받는다는 것을 체감할 수 있게 하기**   
- **로스리더(loss-leader)** 를 통해 많은 고객들이 우테코 식당에 방문하도록 장려하기      
- 내년 1월에 있을 **새해 이벤트**에도 참여할 수 있도록 **12월 방문 고객들 유치하기**

<br>

## 🚀 플래너 기능
**📅 방문 예정 날짜** 와  **🍔 주문 메뉴** 를 통해 아래 목록들을 확인해볼 수 있습니다.

|목록|
|---|
|주문 메뉴 목록|
|할인 전 총 주문 금액|
|증정 메뉴|
|**혜택 내역**|
|총혜택 금액 = 할인 금액의 합계 + 증정 메뉴의 가격(25,000원)|
|할인 후 예상 결제 금액 = 할인 전 총주문 금액 - 할인 금액|
|**12월 이벤트 배지**|

<br>

## 🎁 혜택 내역 세부사항
- 혜택 내역은 아래의 5가지 이벤트로 구성되어 있습니다.      
- 이벤트 조건을 만족한다면 모든 혜택은 **중복적용**이 됩니다.      
- 단 **총주문 금액 10,000원 이상부터 이벤트가 적용됩니다**.   

|12월 이벤트|설명|조건|이벤트 기간|
|---|---|---|---|
|크리스마스 D-Day 할인|1000 + (날짜 - 1) x 100원 할인|없음|12.01 ~ 12.25| 
|평일 할인|디저트 메뉴 1개당 2,023원 할인|일요일 ~ 목요일 중 방문|12.01 ~ 12.31|
|주말 할인|메인 메뉴 1개당 2,023원 할인|금요일, 토요일 중 방문|12.01 ~ 12.31|
|특별 할인|1000원 할인|일요일, 25일(크리스마스 당일) 중 방문|12.01 ~ 12.31|
|증정 이벤트|샴페인 1개 증정|총주문 금액이 12만원 이상|12.01 ~ 12.31|

|---|---|
|별|5천 원 이상|
|트리|1만 원 이상|

- **음료만 주문** 시, 주문할 수 없습니다. 
```
레드와인-3,제로콜라-10,샴페인-5
Error: [ERROR] 음료만 주문 시, 주문할 수 없습니다.
```

- 메뉴는 한 번에 **최대 20개까지만** 주문할 수 있습니다.
```
티본스테이크-10,아이스크림-5,레드와인-10
Error: [ERROR] 메뉴는 한 번에 최대 20개까지만 주문할 수 있습니다.
(e.g. 시저샐러드-1, 티본스테이크-1, 크리스마스파스타-1, 제로콜라-3, 아이스크림-1의 총개수는 7개)
```
- **메뉴에 없는 음식**은 주문할 수 없습니다.
```
비비큐치킨-1,제로콜라-1
Error: [ERROR] 유효하지 않은 주문입니다. 다시 입력해 주세요.
```
- 메뉴의 개수는 **최소 1개 이상**이여야 합니다.
```

Error: [ERROR] 유효하지 않은 주문입니다. 다시 입력해 주세요.
```
- 메뉴가 **중복**돼선 안 됩니다.
```

```


## ⛓️ 플래너 기능 확장

- 메뉴의 확장에 열려있습니다.
- 언제든 메뉴에 음식을 추가하거나 새로운 타입의 메뉴를 만들 수 있습니다.

```javascript
... 

...

export const APPETIZER = Object.freeze({
  '양송이수프' : 6000,
  '타파스' : 5500,
  '시저샐러드' : 8000,
  'type' : 'appetizer',
});

export const MENU = [APPETIZER, MAIN, DESERT, DRINK];

export const NAME = Object.freeze({
  type : 'type',
  appetizer : 'appetizer',
  main : 'main',
  desert : 'desert',
  drink : 'drink',
});
```
<br>

- 혜택 내역은 `map`에 저장되서 관리됩니다.
- `.set`을 통해 언제든 새로운 이벤트를 추가하거나 제외할 수 있습니다.
- 이벤트 출력 순서도 언제든지 변경 가능합니다.

```javascript
  benefitList() {
    const benefits = new Map();

    return benefits
      .set(EVENT_NAME.christmas, this.#event.christmasDDayDiscount())
      .set(EVENT_NAME.weekday, this.#event.weekdayDiscount(this.#cashier.totalTypeQuantity(NAME.desert)))
      .set(EVENT_NAME.weekend, this.#event.weekendDiscount(this.#cashier.totalTypeQuantity(NAME.main)))
      .set(EVENT_NAME.special, this.#event.specialDiscount())
      .set(EVENT_NAME.freebie, this.#event.freebieDiscount(this.#cashier.totalFoodsPrice()))
      .set(EVENT_NAME.condition, this.#event.condition(this.#cashier.totalFoodsPrice()));
  }

```

<br>

- 이벤트 규칙의 변경에 자유롭습니다.   
- 예를 들어, 평일에 디저트가 아닌 에피타이저의 수량에 따른 할인적용을 할 수 있습니다.

```javascript
  totalTypeQuantity(foodType) {
    return this.#foods
      .filter((food) => food.type() === foodType)
      .reduce((acc, cur) => acc + cur.quantity(), 0);
  }

```

<br>

- 각 이벤트의 일정을 언제든 변경 및 삭제할 수 있습니다.

```javascript
  #week() {
    const WEEK = ['SU', 'MO', 'TU', 'WE', 'TH', 'FR', 'SA'];
    
    return WEEK[new Date(2023, 11, this.#date).getDay()];
  }

```

## ⚒️ 도메인 로직

- Event : 이벤트 적용 총괄, 날짜 유효성 검증.

- Food : 하나의 음식에 관련된 정보 취합 및 배포, 음식 이름 유효성 검증.

- Cashier : Food를 바탕으로 음식들의 정보 취합, 주문 유효성 검증.

- Supervisor : Event 와 Cashier 를 통해 혜택내역 정보 총괄.
