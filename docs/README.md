<요구 사항 - 페어 매칭>
- 같은 레벨 동안은 같은 페어를 만나지 않는다.
- 위를 위반시 크루 목록을 다시 shuffle 후 매칭
- 3회 시도까지 매칭이 되지 않거나 매칭을 할 수 있는 경우의 수가 없으면 throw Error
- 구현 방법 : Shuffle 로 섞는다 -> 앞에서부터 순서대로 두명씩 페어 -> 홀수면 마지막크로는 마지막 페어포함
